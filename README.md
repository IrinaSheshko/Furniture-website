# 🏠 Home&You — Landing Page

Элегантная посадочная страница для магазина интерьерного декора. Проект сфокусирован на чистом дизайне, эстетике и удобной навигации.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Furniture-website/)

---

### 🌟 Особенности проекта:
* **Адаптивная витрина:** Сетка товаров реализована с учетом современных стандартов верстки.
* **Эстетичный дизайн:** Использование шрифта *Cormorant Garamond* и качественных изображений с Unsplash для создания премиальной атмосферы.
* **Чистый код:** Проект написан на чистом HTML5, CSS3 и JavaScript без использования тяжелых фреймворков.
* **Интерактивные элементы:** Подключен скрипт для обработки действий пользователя (поиск, навигация).

---

### 🛠 Стек технологий:
* **HTML5:** Семантическая верстка.
* **CSS3:** Работа с Flexbox/Grid, кастомные шрифты и позиционирование элементов.
* **Vanilla JavaScript:** Логика взаимодействия на странице.
